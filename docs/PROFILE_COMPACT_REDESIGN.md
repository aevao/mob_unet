# Компактный переработанный профиль

## Проблемы предыдущей версии

1. **Аватар слишком большой** - 128px перетягивал внимание
2. **Имя разбито на несколько строк** - плохая читаемость
3. **Экран выглядел пустым** - большие отступы, несобранная композиция
4. **Нет чёткой композиции** - элементы разбросаны
5. **Кнопка выхода доминирует** - агрессивный красный цвет

## Новое решение

### 1. Компактный Header (ProfileHeaderCompact)

**Размеры:**
- Высота: ~80px (было ~200px)
- Аватар: 56px (было 128px) - уменьшен на 56%
- Имя: в одну строку, text-lg, font-semibold
- Роль: вторичный текст, text-xs

**Композиция:**
```
[Аватар 56px] [Имя + Роль] [Кнопка редактирования]
```

**Преимущества:**
- ✅ Компактность - занимает минимум места
- ✅ Вся информация в одну строку
- ✅ Чёткая иерархия - имя крупнее, роль меньше
- ✅ Кнопка редактирования рядом, не отдельно

### 2. Список данных (ProfileList)

**Структура:**
- Убрана карточка с 2 полями
- Единый список с разделителями
- Формат: [Иконка] [Label] [Значение]

**Основные поля (всегда видны):**
- Телефон
- Дата рождения

**Вторичные поля (скрыты по умолчанию):**
- Имя, Фамилия, Отчество
- Email
- Номер студенческого билета

**Преимущества:**
- ✅ Минимализм - только важное видно сразу
- ✅ Единая структура - все элементы одинаковые
- ✅ Легко сканировать - иконка + текст
- ✅ Оптимизация пространства - второстепенное скрыто

### 3. Кнопка выхода (ProfileActionButton)

**Изменения:**
- Убрана агрессивная красная подсветка
- Нейтральный серый цвет
- Компактный размер
- Расположена внизу экрана

**Преимущества:**
- ✅ Не доминирует визуально
- ✅ Продуктовый подход - не акцент на выходе
- ✅ Взрослый дизайн - спокойные цвета

## Технические детали

### Компоненты

**ProfileHeaderCompact.tsx**
```typescript
- Аватар: 56px (h-14 w-14)
- Имя: text-lg, font-semibold, numberOfLines={1}
- Роль: text-xs, вторичный цвет
- Кнопка редактирования: 18px иконка
```

**ProfileList.tsx**
```typescript
- ListItem: иконка 20px + label + value
- Разделители: 1px, отступ слева 56px
- Кнопка "Показать еще": центрированная, акцентный цвет
```

**ProfileActionButton.tsx**
```typescript
- Нейтральные цвета (серый)
- Компактный padding
- Прозрачный фон с border
```

### Отступы и сетка

- **Горизонтальные**: 16px (px-4)
- **Вертикальные**: 12px (py-3) для элементов списка
- **Разделители**: 1px, отступ слева 56px (выравнивание с иконками)

### Цветовая схема

**Light Mode:**
- Текст: #111827 (gray-900)
- Вторичный текст: #6B7280 (gray-500)
- Иконки: #6B7280 (gray-500)
- Разделители: #E5E7EB (gray-200)

**Dark Mode:**
- Текст: #E5E7EB (gray-200)
- Вторичный текст: #9CA3AF (gray-400)
- Иконки: #9CA3AF (gray-400)
- Разделители: #1F2937 (gray-800)

## Сравнение

| Параметр | Старая версия | Новая версия |
|----------|---------------|--------------|
| Высота header | ~200px | ~80px |
| Размер аватара | 128px | 56px |
| Имя | 2-3 строки | 1 строка |
| Отступы | Большие | Компактные |
| Кнопка выхода | Агрессивная | Нейтральная |
| Визуальный вес | Высокий | Сбалансированный |

## Результат

✅ **Компактность** - занимает меньше места
✅ **Чёткая иерархия** - имя крупнее, роль меньше
✅ **Минимализм** - только важное видно сразу
✅ **Продуктовый подход** - не "анкета", а профиль
✅ **Визуальный баланс** - ничего не доминирует
✅ **Взрослый дизайн** - спокойные цвета, профессиональный вид
